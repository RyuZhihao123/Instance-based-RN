Selected Fun Projects (娱乐项目)
Non-Photorealistic Rendering. 2016

I implemented several NPR effects with OpenGL, including Pencil and ink-water styles etc., with reference to a popular book titled 'real-time rendering'. [video]
LowPoly-style image generator. 2018

I tried to reproduce a journal paper published in the Visual Computer [Gai et al.] during my undergraduate time. It translates the input images into the popular low-polygon fashion style. [video]

Sketch-based plant modeling system. 2016

This system is my attempt to reproduce the basic functions of a Eurographics article [Okabe et al.] using OpenGL. It enables the users to create 3D plant models through sketching several 2D skeletal strokes. [video]
Face detection via cascade AdaBoost classifier. 2018

Despite the fact that deep learning methods like Mask R-CNN greatly facilitate detecting objects from photos. However, conventional algorithms that are devised more mathmatically also deserve our study. This program achieved realtime face detection via cascade AdaBoost in C++ (no use of OpenCV), with reference to [Viola et al.] published in CVPR. (unfortunately, only my face was not detected accurately).
Simple Minecraft in 3 days using OpenGL! 2017

I tried to build a simple but nice looking Minecraft using OpenGL within 3 days. I implemented the endless terrain and tried to render it with high performance (average in 150 FPS). Also, people are able to freely construct their own buildings. By the way, all the textures I used were designed by myself from scratch using PhotoShop Haha.
SVM classification via sequential minimal optimization. 2017

This SVM visualization system was programmed using C++ and Qt, which was originally used for my Numerical Optimization course at UCAS. The binary classification was solved by Sequential Minimal Optimization (SMO). [video]
